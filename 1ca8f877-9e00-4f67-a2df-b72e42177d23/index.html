0061 习惯打卡-DataView

实现了习惯打卡功能，效果如下：


详细代码请见： https://github.com/dmscode/Obsidian-Templates#习惯打卡。

现在有了代码仓库，在这边需要书写的介绍就很少了。这边会侧重写我的思考。

一个方向

在纸质笔记上，我使用的笔记方法是“子弹笔记”（Bullet Journal）。但纸质笔记荒废已久，然后发现不做笔记（日记）日子过去了就仿佛没有存在过，过的毫无真实感。真就个浑浑噩噩，宛如行尸走肉一般。

所以，记录，这件事情还得坚持。

所以，打算把子弹笔记的方法搬到 Obsidian 上面来。

本来想做标记列表的实现的，结果发现 Minimal 已经实现得非常好了，实在没必要画蛇添足。

然后就做了打卡，记录一些日常。后面还会有年视图和月视图等。

附录：Minimal 内置标记

做个列表供大家参考吧，我用文字描述一下图标，尽量不做用途上的对应

  • [*] : 五角星
  • [-] : 整行删除
  • [<] : 日历
  • [>] : 向右飞机
  • [I] : 发光灯泡
  • [b] : 书签
  • [c] : 向下竖拇指
  • [d] : 指数下跌箭头
  • [f] : 火焰
  • [k] : 钥匙
  • [l] : 地标
  • [p] : 向上竖大拇指
  • [u] : 指数上涨箭头
  • [w] : 生日蛋糕
  • [?] : 问号
  • [/] : 圆形，半黑半白
