杂志起始页视图

这是一个自用视图，一些细节并未设置为可修改。

调用方法：

dv.view('Start', {
  navs: {
    基础页面: [
      '🏠Start|Start.md',
      '📇Index|Index.md',
      `📝Memos|Life/Logs/${new Date().getFullYear()}.md`,
      '⛳Sandbox|Inbox/Sandbox.md',
    ],
    生活列表: [
      '💳刷卡记录|Life/List/刷卡记录.md',
      '💰缴费记录|Life/List/缴费记录.md',
      '🛒购物清单|Life/List/购物清单.md',
      '📰服务订阅|Life/List/服务订阅.md',
    ]
  }
})

选项中可以给出多个导航组，每个链接的名称和地址用 |分隔。

下面生活缴费记录是读取 Life/List/缴费记录.md 中的内容，将每个 Section 下第一个时间作为最新缴费时间，计算和当前的间隔天数。

故事

本来这个视图很简单，用 dataview 的的方法渲染 callouts 就行了，还可以折叠，然后写好了，测试完了。结果第二天一更新 Dataview 发现这个方法失效了……大无语，算了，自己写元素代码吧，成功了，很好，终于结束了，第二天一更新 Dataview，嘿，用它插入的元素不能自定义代码了……大无语，插入元素的功能也自己写吧。写完发现，Dataview 显然也认识到了这些问题，全都更新修复了。

哼唧！
