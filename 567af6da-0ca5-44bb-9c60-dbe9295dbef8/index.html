0000 原本无一物

我不喜欢笔记，我以前从来不记笔记，我的记忆就是我的笔记。

但是，记忆这东西……有一天会叛逃，靠！釜底抽薪了这是

不想讲知识管理啥的，没那么高大上，我管理的有些不是知识，我只是不想忘记钥匙放哪里了。

为啥是 Obsidian？

都行，我真没所谓，记事本也行，只不过各种意义上的不好用罢了。VS Code 也行，Typora 也行……我还能随口讲几十个（摸摸脑袋，好像又没记住那么多名字）。

不过这里涉及到了我笔记的一个重要原则：绝不依赖特定平台。免得我把我的知识啊，人生啊，一切的一切写在上面的，以为自己忘不掉了，然后它收费了，或者倒闭了。我以为笔记是我的第二大脑，哪承想第二大脑也叛逃了。

所以我的笔记是 Markdown 格式，本质是文本文件，用标记添加简单的格式，用记事本也能读，而且可读性还不错。想怎么存储怎么存储，想怎么同步怎么同步。

至于 Obsidian，就是个管理/编辑工具而已，随时可以更换，甚至我现在日常也用它和其他编辑器配合。

行了，结帖。

Obsidian 香啊！

无论是 Dataview 插件还是 Templater 插件，都可以自己写 JS，恰好，我会写 JS。巧么了，这不是。

都能自己写脚本了，而且还是我会的语言，那就约等于没啥学习成本，还拥有最大的自由度。我当然觉得香了。

这个系列，只是从我的角度记录我的使用，分享其中的舒爽和乐趣。还有一些插件用法和我写的小脚本。
