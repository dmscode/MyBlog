标签说明：

关于设备

因为我只有 Windows 和 Android 设备，所以记录的主要是这两个平台的软件。因为有些软件是多平台的，所以索性把两个平台的软件都写在了一起

软件

浏览器

  • Chrome - 当前浏览器的事实王者，很多浏览器都是基于它的开源项目打造的 Chromiu #FullPlatform
  • Edge - 微软的新浏览器，和系统捆绑强推，相比 Chrome 增加了许多实用功能，但也强制塞给了用户许多用不上的功能。 #FullPlatform
  • Firefox大名鼎鼎的火狐，因为开源，有很多衍生版，很不错。定制性非常强，但是受到 Chrome 冲击，用户更少了。 #FullPlatform #OpenSource
  • Opera - 浏览器界的先驱者，很多现在觉得习以为常的功能都是由它发明启用的。非常小众，被 360 收购之后变得更加低调无声。 #FullPlatform
  • Vivaldi - Opera 的创始人之一开发的新浏览器，有很多有趣的功能，但是 WebUI 对电脑性能有一定要求。 #FullPlatform
  • Brave - 一个主打隐私安全，内置区块链钱包的浏览器。有一点特色。 #FullPlatform
  • CentBrowser - 百分浏览器，一个常见的定制化 Chromium，但有时候会随缘停更。 #Windows
  • Waterfox - 水狐，一个第三方编译版 Firefox，有增强，以前以 64 位版本出名，现在 Firefox 也有 64 位了。 #Computer
  • Via - 一款非常小巧的手机浏览器，很知名。 #Android #iOS
  • X浏览器 - 同样非常小巧的手机浏览器，内置用户脚本支持。 #Android
  • Alook - 有非常多实用功能的浏览器，内置用户脚本支持。 #Android #iOS
  • Soul Browser有非常多使用功能的浏览器，会替换网页视频的播放器，尽量统一播放体验，并可下载在线视频。 #Android
  • Arc - 一款新型浏览器,可根据用户使用习惯进行自适应调整。它提供了诸如 Spaces、Profiles、Split View、Themes 和隐私保护等功能,帮助用户组织和定制在线体验。 #FullPlatform

编辑器

  • Visual Studio Code - 微软出品，强大，简单，一般需求用它都够了。 #Computer #Web
  • Sublime Text - 很强大的编辑器，收费软件。 #Computer #Paid
  • vim - 知名的命令行编辑器，侧重命令，可以全键盘操控。 #Computer
  • Neovim - Vim 的一个流行分支。
  • SpaceVim - Vim 的一个流行分支。
  • Emacs - 知名编辑器，侧重快捷键，其 Org mode是非常优秀的笔记形式。 #Computer
  • Notepad++ - 很不错很小巧的编辑器，作者中国台湾人 #Windows
  • EmEditor - 很强大、小巧的编辑器，历史悠久。 #Windows #Paid
  • EditPlus - 很强大、小巧的编辑器，历史悠久。 #Windows #Paid

Markdown 编辑器

  • Typora - 个人以为在可视化编辑的体验目前无出其右。 #Computer

笔记

  • Obsidian - 基于 Markdown 文件的笔记软件，用户自由度非常高。 #FullPlatform
  • Logseq - 基于数据库的笔记软件，开源。 #FullPlatform #OpenSource
  • Notion - 网络笔记，基于“块”的概念。 #FullPlatform

下载

  • IDM - 知名下载器，唯一的缺点是收费。 #Windows #Paid
  • FDM - 免费下载器。 #Computer #Android
  • Aria2 - 强大的命令行下载工具。 #Computer
  • Motrix - 基于 Aria2 的下载器，有好看的 GUI。 #Computer
  • AriaNg - Aria2 的 Web GUI。
  • Aria2e - Aria2 的管理扩展，基于 AriaNg。 #Chrome
  • ADM - Android 下的一款强大的下载工具。 #Android

同步

  • GoodSync - 功能非常完善的同步软件，唯一的缺点是收费，现在还是订阅制。 #FullPlatform #Paid
  • FreeFileSync - 很多时候被当做上面的免费替代品，其实功能差距很明显，优点是免费。 #Computer
  • Syncthing - 设备间文件同步工具，命令行，有 WebUI。 #Computer #Android

存储映射

  • AList - 映射各种存储，无同步功能，但可以引用各种下载程序（aria2、qBittorrent），WebUI，可提供 webdav 服务。 #Computer
  • Rclone - 可以将各种存储设备映射为本地磁盘，也可以在其间进行同步。 #Computer
  • RaiDrive - 将各种网络存储映射为本地磁盘，不过现在收费了。 #Computer #Paid
  • Mountain Duck - 将各种网络存储映射为本地磁盘，收费。 #Windows #Mac #Paid

文件备份

  • Kopia - 开源命令行（有 GUI）文件备份工具，可映射存储，可存储文件快照。 #Computer
  • Duplicati - 开源命令行（有 WebUI）文件备份工具。 #Computer
  • BorgBackup - Linux 下的命令行文件备份工具，有 GUI Vorta #Mac #Linux

媒体

  • Potplayer - 简单易用又十分强大的播放器，不过目前被加入了广告，虽然不明显。 #Windows
  • VLC - 功能强大的播放器，但是界面朴素。 #FullPlatform
  • mpv - 据说很强大的播放器，但我调教不出来，一直用不习惯。有各种修改版可供选择。 #Computer
  • NewPipe - Youtube 的第三方下载工具，支持订阅（无需登录） #Android #OpenSource
  • youtube-dl - 命令行的在线视频下载工具，有很多基于它的 GUI 版本。 #Computer

阅读

  • 阅读 - 自添加源的网络小说阅读器，也可以用来听有声书，看漫画。 #Android #OpenSource
  • 墨阅 - 一款稍后阅读软件，也可以用作剪藏。 #Android
  • Calibre - 强大的老牌电子书管理转换软件。 #FullPlatform

美化

  • Wallpaper Engine - 动态壁纸，支持将视频、网页设置为壁纸，可玩性非常高。 #Windows #Android #Paid
  • Rainmeter - 知名老牌美化工具，可以理解为极度灵活的桌面挂件，比较小众。 #Windows
  • TaskBarHider - 状态栏隐藏工具。 #Windows
  • TranslucentTB - 任务栏透明。 #Windows

增强

  • PowerToys - 微软小工具集，很实用，安装体积大，但是运行资源占用并不高。 #Windows
  • uTools - 搜索框式快速启动工具，支持插件，可以做的事情非常多。高级功能需购买会员。 #Windows #Paid
  • Listary - 搜索框式快速启动工具，有时候很久不更新。高级版收费。 #Windows #Paid
  • Capsicain - 驱动级键盘映射工具。 #Windows
  • AutoHotkey - 一种编程语言，可以非常简单的进行键盘映射，宏编写等。 #Windows
  • Everything - 速度超快的全盘文件搜索工具。 #Windows
  • Flow Launcher - 一款用于 Windows 的快速文件搜索和应用程序启动器。 #Windows

文件管理

  • Files - 一个第三方文件管理器，界面美观，支持多标签，分栏，Git，不过速度略慢
  • OneCommander - 适用于 Windows 11 和 Windows 10 的现代文件管理器
  • Total Commander - 老牌专业资源管理器
  • Directory Opus - 这个比较容易上手
  • XYplorer - 我用不惯
  • Q-Dir 分屏能力强大，但是我屏幕一般不够它发挥。按钮较小，对我的视力不友好

Chrome 扩展

  • 标签分组扩展 - 可以设定规则让标签页自动分组
  • 沉浸式翻译 - 翻译体验最好的插件
  • 篡改猴 - 油猴子
  • 读点东西 - 阅读模式
  • 二管家 - 扩展管理
  • Aria2 Explorer - 下载管理
  • floccus bookmarks sync - 书签同步
  • JSON Viewer - 日常看资源方便点
  • KeePassXC-Browser - 密码管理
  • Smart TOC - 网页内容目录
  • Vimium - 在网页中使用 Vim 快捷键
  • Volume Master - 音量控制器 - 音量控制

VS Code 插件

  • Align by RegEx - 通过正则表达式对选定的文本行进行对齐。
  • AutoHotkey Plus - 为 VS Code 提供 AutoHotkey 语言支持。
  • AutoHotkey v2 Language Support - 基于 vscode-lsp 的 AutoHotkey v2 语言支持。
  • Beancount - 用于 Beancount 的 VSCode 扩展。
  • Code Runner - 运行 C、C++、Java、JS、PHP、Python、Perl、Ruby、Go、Lua、Groovy、PowerShell、CMD、BASH、F#、C#、VBScript、TypeScript、CoffeeScript、Scala、Swift、Julia、Crystal、OCaml、R、AppleScript、Elixir、VB.NET、Clojure、Haxe、Obj-C、Rust、Racket、Scheme、AutoHotkey、AutoIt、Kotlin、Dart、Pascal、Haskell、Nim 等语言的代码。
  • Codeium: AI Coding Autocomplete and Chat for Python, Javascript, Typescript, Java, Go, and more - 现代编码超能力:适用于您最喜欢的语言的免费 AI 代码加速插件。输入更少,编码更多,更快发布。
  • Css Sorting - VS Code CSS 排序插件,用于按指定顺序对 CSS 规则内容进行排序。
  • CSS-in-JS - CSS-in-JS 自动完成和转换。
  • Document This - 自动为 TypeScript 和 JavaScript 文件生成详细的 JSDoc 注释。
  • eta-vscode - Eta 模板语言的语言支持。
  • gitignore - 添加文件到 .gitignore。
  • HTML CSS Support - HTML 的 CSS 智能感知。
  • Increment Selection - 使用多个光标递增选择。
  • Live Server - 为静态和动态页面启动具有实时重载功能的本地开发服务器。
  • Markdown All in One - 编写 Markdown 所需的一切(快捷键、目录、自动预览等)。
  • Material Icon Theme - Visual Studio Code 的 Material Design 图标。
  • Mermaid Markdown Syntax Highlighting - 为 Mermaid 图表语言提供 Markdown 语法支持。
  • One Dark Pro - Atom 经典的 One Dark 主题for Visual Studio Code。
  • Outline Map - 一个可视化、交互式的大纲地图,结合了大纲的清晰性和小地图的直观概览。增强版的 vscode 内置大纲。
  • Path Intellisense - 自动完成文件名的 Visual Studio Code 插件。
  • Prettier - Code formatter - 使用 prettier 进行代码格式化。
  • SCSS IntelliSense - 为 SCSS 提供高级自动完成和重构支持。
  • String To Any Case - 将选定的字符串从一种大小写格式转换为另一种。支持的格式有...
  • Todo Tree - 在树视图中显示 TODO、FIXME 等注释标签。
  • Trailing Spaces - 高亮显示尾随空格并立即删除它们!
  • Vue - Official - Vue 的语言支持。
  • Word Count CJK - 支持 CJK 语言的字数统计扩展。
  • AWS Toolkit - 包括 CodeCatalyst、Application Composer,以及对 Lambda、S3、CloudWatch Logs、CloudFormation 等多项服务的支持。
  • Foam - 用于笔记和知识库的 VS Code + Markdown + Wikilinks。
  • Instant Markdown - 即时预览您的 Markdown 文件。
  • koroFileHeader - 用于生成文件头部注释和函数注释的插件,支持所有主流语言,功能强大,灵活方便,文档齐全,使用简单。

Obsidian 插件

  • Excalidraw - 自由且强大的白板绘图工具。
  • Dataview - 高级查询功能，可以使用 JS 自己书写查询视图。
  • Templater - 更强大的模板。
  • Tasks - 跟踪整个库中的任务。支持到期日期、循环任务、完成日期、检查列表项子集和过滤。
  • Kanban - 创建基于 Markdown 的看板。
  • Style Settings - 调整主题、插件和代码段的 CSS 变量。
  • Minimal Theme Settings - 控制 Minimal 主题的颜色和字体。
  • Outliner - 像在 Workflowy 或 Roam Research 中一样处理列表。
  • QuickAdd - 快速添加新笔记或内容到你的库。
  • Advanced URI - 使用 URI 控制一切。
  • Quick Switcher++ - 增强版快速切换器,搜索打开的面板和符号。
  • File Explorer Note Count - 在文件资源管理器中查看每个文件夹中的笔记数量。
  • Plugin Update Tracker - 了解已安装插件何时有更新,并评估升级的风险。
  • Paste image rename - 粘贴图像后重命名,支持命名模式和自动重命名。
  • Local images plus - 本地图像的重生版本,将 Markdown 笔记中的图像下载到本地存储。
  • Update time on edit - 保持前 matter 与最后编辑时间同步。注意排除掉模板和插件的文件夹。
  • Mindmap Nextgen - 将笔记预览为 Markmap 思维导图。
  • Archiver - 将已完成的任务移动到带有日期树的归档中。
  • Open vault in VS Code - 在 Visual Studio Code 中打开库的按钮和命令。优点是打开整个文件夹，并定位到对应文件。
  • Share Note - 即时分享/发布笔记,内容和主题完全如同在 Obsidian 中所见。默认情况下,数据是加密共享的,只有你和你发送给的人有密钥。
  • Open with - 添加多个其他程序来打开笔记。优点是针对任何文件都可以从右键菜单中用特定软件打开。
  • Scroll Offset - 保持光标前后的自定义距离。
