有这样一个链接$1

https://greasyfork.org/zh-CN/scripts/373270-%E9%93%BE%E6%8E%A5%E5%9C%B0%E5%9D%80%E6%B4%97%E7%99%BD%E7%99%BD
``$1

很长，显示出来一点也不好看，也看不懂。这是因为里面有一串中文，而中文被编码了，如果变成下面这样就舒服多了，而且具有很好的可读性$1

[https://greasyfork.org/zh-CN/scripts/373270-链接地址洗白白](https://greasyfork.org/zh-CN/scripts/373270-链接地址洗白白$1

当然，如果用这个链接中的脚本，可以把它缩的更短，然而那是另外的角度了$1

## 模板代$1

```eta
<%*
  const urlSource =  await tp.system.clipboard()
  const url = decodeURIComponent(urlSource)
  const title = await tp.system.prompt('输入链接标题')
  tR += `[${title ? title : url}](${url})`
%>
``$1

## 使用方$1

先复制好链接，即链接地址在剪切板里，然后使用模板插入，填写标题，如果不填标题则直接显示地址。然后就好了。
