最近看到一个插件，实现的效果基本上就是添加一个 Dataview 的自定义视图。那这么说我也可以把我写的东西做成插件$1

然后发现了一个小坑。那个插件的 CSS 是长期加载的，函数是一直注册的。稍微对性能有那么一点拖累。毕竟可能这个功能我们只在特定的笔记下才需要使用$1

而 Dataview 本身在这方面优化得很好，自定义视图不使用就不加载。这样我为特殊需求写一堆非常复杂的代码，就没什么压力。而且很方便，很自由。日常的需求也不高，能这么方便好用当然很开心了$1

不是所有插件都有这种问题，只是一种可能$1
