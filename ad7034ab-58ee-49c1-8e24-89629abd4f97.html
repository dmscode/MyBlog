学过

我整理自己的 Git 速查表格时很多人告诉我已经有一个了，然而其实我是先汉化了那个才来做我自己的。当然，大家都不知道嘛。不过接下来想想，无论是我做的这份，还是我汉化的这份，也都没什么人看啦。重要的只是，我自己学过。

遗忘

我想写一点笔记碎片，但是还有一些以前的想法也要写。然后写下了上一条，却忘记了这一条想要写什么。

总结

感觉用 AI 做阶段性笔记总结是个不错的功能，这几天为这个想法而兴奋，终于有时间动手了，列出了自己最近的笔记……算了，不用总结了，毕竟主要是日记，这些东西发送给 AI 真不放心。但除了这些，我做了什么，我一个星期也就整理了 git 笔记而已，一两篇而已，不用总结，我看一眼标题就全想起来了。

笔记

笔记软件挺多的，层出不穷的，但我的笔记是个长期的工程，不希望被那些短期的概念和噱头引导。也许从最开始选择笔记软件本身就是一个错误的问题，因为选择就意味着绑定依赖。

五笔

我又换到五笔了，因为觉得双拼选字太烦了（主要还是搜狗的辅助码用不惯）。但五笔提笔忘字的问题严重阻碍了我提升打字速度，而且用五笔我找不到输入节奏，觉得不定长，而且老在按空格，有点乱乎。

宣传

在小红书上宣传 WinSize，数据很好看，目前三千多阅读，90 红心，140 收藏。但销售量就，就……好像没有，也许有一个？！软件付费这事儿吧，在国内还是有点，就是买网盘会员，买视频会员，买……都能接受，但是软件付费还是不太行。甚至我猜测（没有测试，没有数据，只是胡思乱想的猜测）也许订阅制比买断制更好卖，比如 36 元买断和 3 元 一个月，大概很多人觉得前者贵，而后者则可以买一个月试试。

收藏

那么多人收藏啊，可我这只是个介绍贴，又不是技巧分享，收藏起来真没啥用。我和大家也不熟，应该不会是收藏了哄我开心的。那，收藏是为什么呢？我真的感到莫名。

整理

从上一条发散思维一下，收藏就像过去做笔记的剪藏，就像微博下面成排的 @xx笔记 。但笔记不是光记，是要整理内化的。但是感觉好多人就是收藏吃灰，把一些数据从一个地方搬到另一个地方，但和自己无关。

笔记是要整理的啊！整理是很痛苦的……（想起了前些天整理 git 速查表的痛苦
