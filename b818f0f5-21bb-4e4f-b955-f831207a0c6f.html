就这么个“网红路牌”，这次真的只是个标题，别再填内容了（没写内容部分的样式）。文字不宜过长，主要是到手机上显示不下，这玩意儿还弄个多行显示不至于吧，就图一乐呗。基本上白色文字不超过黑色文字的宽度在一般手机上就不会溢出。对，黑色文字不是广告，是尺（笑

将如下代码保存到 .obsidian/snippets/RoadNameSign.css 文件中（自行创建文件），然后在 Obsidian 设置——外观——代码片段中开启。即可使用 road-name-sign 类型 callouts 创建这种堆叠卡片了。

 * 网红路牌

 * @created: 2024-05-14
 * @updated: 2024-05-14
 * @version: 0.0.1
 */

  --callout-color: 0, 51, 255;
  --callout-text-color: #ebeced;
  --callout-icon:  arrow-big-left;
  background: transparent;
  padding: 20px;
}
.callout[data-callout="road-name-sign"] > .callout-title {
  position: relative;
  background: rgb(var(--callout-color));
  border-radius: .5rem;
  border: 3px solid var(--callout-text-color);
  box-shadow: 0 0 0 5px rgb(var(--callout-color));
  margin: 0 auto 1.6rem;
  padding: 0;
  max-width: 720px;
}
.callout[data-callout="road-name-sign"] > .callout-title::before {

  display: none;
