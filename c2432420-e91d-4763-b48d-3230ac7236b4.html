私密文字

让文字隐藏起来，仅鼠标悬停可见，我的银行卡密码是：怎么可能告诉你呢。

给文档添加属性：cssclasses: secret，然后把文档中需要隐藏的文字高亮：==需要隐藏的文字==，在 CSS 片段中添加如下代码并启用。

/* ======== 文字模糊 ======== */
.secret span.cm-highlight,
.secret mark {
  background-color: rgba(0, 0, 0, 0);
  color: transparent;
  text-shadow: 0 0 1em var(--text-normal);
}
.secret .cm-active:hover span.cm-highlight,
.secret span.cm-highlight:hover,
.secret mark:hover {
  color: var(--text-normal);
  text-shadow: 0 0 0 transparent;
}
