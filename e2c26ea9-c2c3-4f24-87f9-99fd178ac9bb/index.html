估计看到这里你们已经忘记我在分享前做的设定了：CSS 目录指的是“设置——外观——CSS 代码片段”中所指的路径 .obsidian/snippets 。所以我贴心的再说一次$1

在这个目录（没有就自己新建文件夹）下创建 Active-Tab-Border.css 文件，然后里面粘贴如下内容$1

.workspace-tab-header.is-active {
  border-bottom: 2px solid #668899;
}
``$1

保存好，没任何效果，很正常。到：“设置——外观——CSS 代码片段”位置看一下，是不是提示有一个同名代码片段了？开启它就好了。如果没有点一下那个刷新按钮。如果还没有，那你上面的什么步骤搞错了$1

效果是激活的标签页下面有个深蓝色的下划线，这样比较好找，就这$1

不过这已经足够证明自定义样式的便捷性了，只要你会写 CSS，这软件长什么样完全你说了算。对，这就是写主题。不过不用写完整主题，而是在当前状态下打补丁，也挺好玩的。
