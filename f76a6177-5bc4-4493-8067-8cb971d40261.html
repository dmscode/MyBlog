Obsidian 默认快捷键

沙盘，开启所有核心插件，全默认。

软件操作

名称 快捷键
查看帮助 F1
打开设置 Ctrl + ,
关闭 Ctrl + Shift + W
切换编辑/预览视图 Ctrl + E

文件操作

名称 快捷键
保存当前文件 Ctrl + S
编辑文件名 F2
新建笔记 Ctrl + N
在右侧创建笔记 Ctrl + Shift + N

编辑

名称 快捷键
插入链接 Ctrl + K
加粗/取消加粗 Ctrl + B
倾斜/取消倾斜 Ctrl + I
删除段落 Ctrl + D
注释/取消注释 Ctrl + /
切换待办事项状态 Ctrl + L
在当前笔记中查找 Ctrl + F
在当前笔记中查找并替换 Ctrl + H
增加文档属性 Ctrl + ;

链接导航

名称 快捷键
打开光标处的链接 Alt + Enter
返回 Ctrl + Alt + ←
前进 Ctrl + Alt + →
在新窗口中打开光标处链接 Ctrl + Alt + Shift + Enter
在新面板中打开光标处链接 Ctrl + Alt + Enter

标签列表

名称 快捷键
关闭当前标签页 Ctrl + W
新标签页 Ctrl + T
在新标签页中打开光标处链接 Ctrl + Enter
重新打开标签页 Ctrl + Shift + T
转到标签 #1 Ctrl + 1
转到标签 #2 Ctrl + 2
转到标签 #3 Ctrl + 3
转到标签 #4 Ctrl + 4
转到标签 #5 Ctrl + 5
转到标签 #6 Ctrl + 6
转到标签 #7 Ctrl + 7
转到标签 #8 Ctrl + 8
转到上一个标签 Ctrl + Shift + Tab
Ctrl + Page Up
转到下一个标签 Ctrl + Tab
Ctrl + Page Down
转到最后一个标签 Ctrl + 9

关系图谱

名称 快捷键
关系图谱: 查看关系图谱 Ctrl + G

快速切换

名称 快捷键
快速切换: 打开快速切换 Ctrl + O

命令面板

名称 快捷键
命令面板: 打开命令面板 Ctrl + P

搜索

名称 快捷键
搜索: 在所有文件中搜索 Ctrl + Shift + F
