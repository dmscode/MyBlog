0050 一键发布博客

各种静态博客生成器用的都是 Markdown 格式的文件，如果把这些内容引入 Obsidian 进行管理是十分合理的。

用 mklink 把笔记中的某个文件夹软连接到博客生成器的文章文件夹。然后用 Shell commands 或者 QuickAdd 运行博客构建与部署命令即可。
