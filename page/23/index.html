GoodSync

虽然有很多同步工具，但某些情况下只能使用特定协议。而在特定协议下，大部分能找到，还算好用的工具都有各种限制，需要付费解锁。

Free Download Manager

一个下载工具，毕竟 Internet Download Manager (IDM) 好用但是要钱。下载需求也不大，用这个免费的来替代。

差别是有的，但各方面我都觉得可以接受，而且颜值不错，日常使用足矣。
【地址】： https://www.freedownloadmanager.org/

Ditto

老牌剪切板管理工具，至今还在坚持更新，可以在 Windows 应用商店下载。

支持分组、局域网同步等等功能，好用，资源占用少，是我装机必备软件之一。

虽然 Windows 10 已经自带了 Win+V 剪切板管理，但是和 Ditto 相比，自带的简直是个玩具。
【地址】： https://ditto-cp.sourceforge.io/

DaVinci Reslove

一个免费且强大的视频编辑工具，我的老电脑也可以比较流畅的使用。剪辑、调色，样样精通，虽然我只会使用其中极少的功能。

网上人称“达芬奇”，资源很多，应该是个很专业的工具。

支持 Windows、Mac、Linux三大桌面平台。
【地址】： http://www.blackmagicdesign.com/products/davinciresolve/
