Settings Sync

将 VSCode 的配置、扩展、主题、代码片段、图标、快捷键……保存到 Github Gist。

并可以从 Github Gist 上获取已保存内容，来恢复上述所有内容。

还可以设置每次有修改自动备份上去……

从而实现了多设备间的配置同步。用起来十分方便。

【地址】： https://marketplace.visualstudio.com/items?itemName=Shan.code-settings-sync

Alignment

代码对齐工具，可以通过调整空白长度，来让多行代码中的相同部分，比如 “=” 等内容在垂直方向上对齐。来满足强迫症患者。

这种对齐，对于需要进行列编辑的情况，会非常友好。

复杂情况下，效果并不是很完美，需要自己进行详细配置。

【地址】： https://marketplace.visualstudio.com/items?itemName=annsk.alignment
